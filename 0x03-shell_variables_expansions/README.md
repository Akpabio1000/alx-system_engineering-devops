# SHELL, INIT FILES, VARIABLES AND EXPANSIONS

This project focuses on expansions, shell arithmetic, variables,
shell initialization files and alaising.

It will feature the use of the following commands: printenv, set, unset, export, alias, unalias, source, printf and ofcourse, the regular shell commands.

As required, several special characters won't be used, 
also, the following commands won't be used: bc, sed and awk.
